# :whale: V2Ray with WebSocket

## Usages
```bash
docker run -d --restart=always -p 80:80 -e PORT=80 -e UUID=<your-uuid> -e ALTERID=2 -e WSPATH=/css whoim/v2ray-ws:latest
```

## Environments

| NAME    | VALUE   |
| :-----: | :-----: |
| Port    | 80      |
| UUID    | -       |
| AlterID | 2       |
| WS-Path | /css    |
